# 🏥 SmartCare – AI-Based Healthcare Web Application

## 📌 Overview

**SmartCare** is an intelligent, location-aware healthcare web application designed to simplify and digitize access to medical services. It combines Artificial Intelligence and real-time geolocation to provide users with instant healthcare assistance, hospital discovery, and emergency support.

The platform acts as a **personal healthcare companion**, helping users manage their health, find nearby hospitals, and receive AI-based medical suggestions.

---

## 🚀 Key Features

### 👤 User Management

* Secure registration and login system
* Email verification & OTP-based password recovery
* Personal health profile management

### 🌍 Location-Based Services

* Real-time user location detection using Geolocation API
* Nearby hospitals using Overpass API
* Google Maps integration for navigation

### 🤖 AI Health Assistance

* Symptom checker for disease prediction
* AI chatbot for health guidance
* Department & doctor recommendations

### 🏥 Healthcare Services

* Hospital discovery based on location
* Doctor search by department
* Appointment booking system

### 🚨 Emergency Support

* One-tap emergency feature
* Quick access to nearest hospitals

### 🌐 Multilingual Support

* English
* Hindi
* Gujarati
